# Client

React + TypeScript + Vite клиент для `aqsha`.

## Команды

```bash
npm run dev
npm run build
npm run lint
```

## Ключевые папки

- `src/api` - HTTP слой и вызовы backend.
- `src/pages` - страницы (главная, биржа, профиль, публикация, карточка объявления).
- `src/components` - переиспользуемые UI-компоненты.
- `src/lib` - утилиты форматирования и localStorage.
